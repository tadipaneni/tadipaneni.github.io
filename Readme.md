2. **Local Development**
   ```bash

   cd tadipaneni.github.io

   git add .
   
   # Commit changes
   git commit -m "Initial portfolio commit"
   
   # Push to GitHub
   git push -u origin main
   ```

   - Source: Deploy from a branch
   - Branch: main, /(root)
